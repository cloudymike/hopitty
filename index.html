    Welcome to Hopitty, the fully automatic personal brewery

    The goal of this project is to create an automated brewing setup for all grain brewing. The software is mainly python code, that drives the brewing hardware, through USB interfaces. The user interfaces is web bases, so you can check on your brew from any computer, smart phone etc. The recipes can be read directly from the BeerSmith software or entered as json files.

    To dive into the code, check out the API documentation. Or go to the gihub pages for more details on the code. Also take a look at the Readme file for more details on how to build and run the software.

    About the Hardware

    The brewing equipment consists of an electric hot water tun, cooler mash tun and and a electric boiler. All sensors and controllers are USB based, this allows flexibility to add and replace equipment. The hardware used while writing this software is described in the hardware wiki pages. All controllers and sensors are USB devices. This enables more flexibility, to use just the devices required. This project started, as example with one USB thermometer and and X10 switch to heat the strike water for the mash, and if that is all you have, the software will still work for this.

    The computer used while developing the software is an Ubuntu (12.04) PC. As the code is in Python it is very portable and and early version was demonstrated working on the RaspberryPI. The USB devices uses code provided by vendors and may be in other languages. All the source code is however included or will be downloaded by the setup script.

    But, does it brew?

    That is the main objective, right? To get an idea of what you can brew with this setup, check out my recipes on the BeerSmith site. An important feature of the software is that it reads the recipe files from BeerSmith software directly. This allows you to focus on creating a great recipe, and then let the software make it for you.
